
## How to use

## Requirement

1. Install libcurl-dev
2. Install lshw

## Usage

1. Replace API_ENDPOINT with your own HTTP API endpoint
2. Compile "gcc -o loli test.c -lcurl"
3. Creating folder "test_folder"
4. Run "php gen.php"
5. Enjoy
